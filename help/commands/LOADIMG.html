LOADIMG imgName, imgFile


Description:

Loads an image file specified by imgFile into the program and assigns it a reference name specified by imgName. Once loaded, the image can be drawn on any window or control by using the DRAWIMG command and unloaded using the UNLOADIMG command. Some image files that can be loaded include .bmp, .jpg, and .gif files.


Usage:

LoadImg "img1", DefPath & "\examples\image1.bmp"

window "win", "Image Test", normal, 100, 100, 400, 300

drawimg "win", "img1", 100, 100

pause
