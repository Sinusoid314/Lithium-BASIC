TIMER name, interval, subName


Description:

Creates a timer that is identified by name. The timer will execute the code within the sub or function specified by subName at intervals specified by interval. The inverval of a timer is measured in milliseconds: 1000 milliseconds is a second, 500 is half a second, etc. The timer can be stoped by calling the STOPTIMER command.


Usage:

showconsol
var hour, min, sec as number

Timer "hourCount", 12000000, HourCount
Timer "minCount", 60000, MinCount
Timer "secCount", 1000, SecCount

pause

sub HourCount
     hour = hour + 1
end sub

sub MinCount
     min = min + 1
end sub

sub SecCount
     sec = sec + 1
     call UpdateDisp
end sub

sub UpdateDisp
     if min = 60 then min = 0
     if sec = 60 then sec = 0
     cls
     print hour & ":" & min & ":" & sec
end sub
