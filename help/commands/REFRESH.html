REFRESH name


Description:

Refreshes the display of the window or control specified by name. Any graphics that were not stuck to the window using the STICK command will be erased from the window when it is refreshed.


Usage:

loadimg "img1", DefPath & "\examples\image1.bmp"

window "win", "Refresh Test", normal, 100, 100, 500, 400
control "draw1", "win", "", drawbox, 10, 10, 100, 100
control "btn1", "win", "Refresh", button, 100, 300, 100, 25

drawimg "draw1", "img1", 10, 10
stick "draw1"
drawimg "draw1", "img1", 40, 40

event "btn1", "click", RefreshWin

pause

sub RefreshWin
     Refresh "draw1"
end sub
