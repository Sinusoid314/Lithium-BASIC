GETSEL name, startVar, endVar


Description:

Returns the starting and ending positions of the text that is currently selected in the textbox or texteditor specified by name. These number values are placed in the variables startVar and endVar.


Usage:

window "win", "Get Selection", normal, 100, 100, 500, 350
control "edit1", "win", "", texteditor, 100, 70, 200, 200
control "btn1", "win", "Get Selection", button, 100, 20, 120, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     var s, e as number
     GetSel "edit1", s, e
     message s & " to " & e, ""
end sub
