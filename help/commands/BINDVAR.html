BINDVAR var1 TO var2


Description:

Binds var1 to var2. Once this is done, the value of var1 will change whenever the value of var2 changes. A variable can have multiple variables bound to it, making all of the bound variables change when its value changes. This cannot work the other way around (when var1's value is changed, it has no affect on var2.) A variable can be unbound from another variable by using the UNBINDVAR command.


Usage:

showconsol

var v1, v2, v3 as string

v1 = "Object"
v2 = "Collection"
v3 = "MetaClass"

print "Before:"
print " " & v1
print " " & v2
print " " & v3
print

BindVar v2 To v1
BindVar v3 To v1

v1 = "nil"

print "After:"
print " " & v1
print " " & v2
print " " & v3


Produces:
Before:
Object
Collection
MetaClass

After:
nil
nil
nil
