SETTEXT name, text


Description:

Sets the text of the window or control specified by name


Usage:

window "win", "SetText Test", normal, 100, 100, 500, 400
control "btn1", "win", "Set Text", button, 100, 100, 150, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     var newText as string
     newText = inputbox("Enter new window text:", "Set Text")
     SetText "win", newText
end sub
