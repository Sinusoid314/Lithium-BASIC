INPUT [[prompt,] variable1] [#handle, variable2]


Description:

Takes in a value from either a file or the user. If prompt and variable1 are specified, the prompt expression is evaluated and printed onto the consol window and the consol waits for the user to enter something. If prompt is omitted, then the consol waits for input without printing a prompt. If #handle and variable2 are specified instead, a value is read into variable2 from the file specified by #handle.


Usage:

showconsol
var entry as string
open "file.txt" for input as #file
Input #file, entry
close #file
print entry
