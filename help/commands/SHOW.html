SHOW name


Description:

Shows the window or control specified by name that has been hidden using the HIDE command.


Usage:

window "win2", "Test Window", normal, 100, 100, 400, 300

window "win1", "Show/Hide Window", normal, 10, 10, 300, 80
control "showBtn", "win1", "Show", button, 10, 10, 100, 25
control "hideBtn", "win1", "Hide", button, 130, 10, 100, 25

event "win1", "close", EndProg
event "showBtn", "click", ShowWin
event "hideBtn", "click", HideWin

pause

sub ShowWin
     Show "win2"
end sub

sub HideWin
     hide "win2"
end sub

sub EndProg
     closewindow "win1"
     closewindow "win2"
end sub
