SPRITE winName, spriteName, x, y, imgName


Description:

Creates a sprite with the label spriteName that will be drawn on the window or control specified by winName. Its initial position is specified by x and y, and its initial image is specified by imgName. This image must have been loaded using the LOADIMG command.


Usage:

loadimg "img1", DefPath & "\examples\sprite1-1.bmp"
loadimg "img2", DefPath & "\examples\sprite1-2.bmp"

window "win", "Sprite Test", normal, 100, 100, 600, 400

event "win", "close", Close

Sprite "win", "sprite1", 10, 10, "img1"
addframe "win", "sprite1", "img2"

timer "timer1", 100, RedrawSprites

pause

sub RedrawSprites
     drawsprites "win"
end sub

sub Close
     unloadimg "img1"
     unloadimg "img2"
     end
end sub
