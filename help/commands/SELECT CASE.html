SELECT CASE expression
     CASE expression
          {code}
     CASE expression
          {code}
     CASE ELSE
          {code}
END SELECT


Description:

Executes a certain block of code based on the given expressions. The SELECT CASE command takes an expression that will be evaluated. Following this can be any number of CASE commands, each with its own expression which is compared to the evaluated expression. If these two expressions match, the {code} directly after the CASE statement is executed. If the evaluated expression does not match any of the CASE expressions, the {code} after the CASE ELSE line is executed. The END SELECT line marks the end of the SELECT CASE block.


Usage:

showconsol
var num as number
input "Enter a number: ", num
Select Case num
     Case 5
          print "Number is 5"
     Case 10
          print "Number is 10"
     Case 20
          print "Number is 20"
     Case Else
          print "Number is not 5, 10, or 20"
End Select
