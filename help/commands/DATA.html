DATA expression, expression, ...


Description:

Allows data to be easily inserted into a program. There can be any number of expressions, and each one can be any type (String or Number). The data can be read into variables durring runtime with the READ command. The data in each DATA statement is local to the sub or function in which it is defined and, unlike variables, code within subs and functions cannot read data from global DATA statements. Variables and arrays should not be used in any data expressions, since all DATA lines are cut out and evaluated before the program starts.


Usage:

Data 4
Data "Andrew", "Dan", "Vince", "Trish"

var nameNum, n as number
var nameStr as string

read nameNum

for n = 1 to nameNum
     read nameStr
     print nameStr
next n
