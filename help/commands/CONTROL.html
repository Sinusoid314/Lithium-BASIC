CONTROL name, parent, text, type, x, y, width, height


Description:

Creates a control labeled name on the window specified by parent. A control is created after its parent window has been created using the WINDOW command. The control's text is specified by text, its position by x and y, and its size by width and height. The type parameter is an identifier specifying the control's type, and can be any one of the following:

     button
     statictext
     texteditor
     textbox
     listbox
     combobox
     drawbox
     picbutton
     checkbox
     radiobutton
     groupbox



Usage:

window "win", "Control Test", normal, 100, 100, 500, 400
Control "btn1", "win", "Button 1", button, 200, 100, 100, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     message "Button 1 was clicked", ""
end sub
