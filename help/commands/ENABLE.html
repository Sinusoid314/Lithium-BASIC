ENABLE name


Description:

Enables the window or control specified by name that has been disabled by the DISABLE command.


Usage:

window "win", "Enable Test", normal, 100, 100, 500, 400
control "btn1", "win", "Enable", button, 200, 100, 150, 25
control "btn2", "win", "Button 2", button, 200, 200, 150, 25

event "btn1", "click", EnableClick

disable "btn2"

pause

sub EnableClick
     Enable "btn2"
end sub
