GETIMG winName, imgName, x, y, width, top


Description:

Creates an image with the name imgName from a section of the window or control specified by winName. The section is specified by x, y, width and height. This image can then be drawn using the DRAWIMG command and unloaded using the UNLOADIMG command.


Usage:

var n as number
loadimg "img1", DefPath & "\examples\image1.bmp"

window "win", "GetImage Test", normal, 100, 100, 400, 300

drawimg "win", "img1", 10, 10

GetImg "win", "img2", 20, 20, 20, 20

for n = 20 to 60 step 10
     drawimg "win", "img2", n, n
next n

pause
