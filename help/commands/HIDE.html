HIDE name


Description:

Hides the window or control specified by name. The SHOW command can be used to display it again. Note of caution: A Lithium BASIC program will continue to run as long as there are windows open in the program. Make sure to close even hidden windows when your program is supposed to terminate, or else it will continue to run in the background of the user's system.


Usage:

window "win2", "Test Window", normal, 100, 100, 400, 300

window "win1", "Show/Hide Window", normal, 10, 10, 300, 80
control "showBtn", "win1", "Show", button, 10, 10, 100, 25
control "hideBtn", "win1", "Hide", button, 130, 10, 100, 25

event "win1", "close", EndProg
event "showBtn", "click", ShowWin
event "hideBtn", "click", HideWin

pause

sub ShowWin
     show "win2"
end sub

sub HideWin
     Hide "win2"
end sub

sub EndProg
     closewindow "win1"
     closewindow "win2"
end sub
