WHILE expression
     {code}
WEND


Description:

Allows a program to execute some code a certain amount of times. If expression is true, the {code} is executed, and will continue to loop around and execute until expression becomes false. Note of caution: Be careful when using this and the FOR...NEXT loop, as it could go into an infinate loop if expression never becomes false.


Usage:

showconsol
var num as number
While num < 100
     num = num + 5
     print num
Wend
