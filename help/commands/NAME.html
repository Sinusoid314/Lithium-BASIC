NAME oldName AS newName


Description:

Renames the file specified by oldName to the file name newName. If the file paths of each are different, the file oldName will be moved to the location of newName.


Usage:

var newName as string
showconsol

open "c:\ff.txt" for output as #file
     print #file, "file data"
close #file

input "Enter a new file name: ", newName

Name "c:\ff.txt" As "c:\" & newName
