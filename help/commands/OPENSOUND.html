OPENSOUND soundName, fileName


Description:

Opens a sound file, specified by fileName, and assigns it a name specified by soundName. Some sound files that can be loaded include .wav and .mid files. Once you are done using the sound, it should be closed using the CLOSESOUND command.


Usage:

showconsol

var v as number

OpenSound "midi", DefPath & "\examples\the guess who - no time.mid"
input "Play>> ", v

playsound "midi"
input "Stop>> ", v

closesound "midi"
