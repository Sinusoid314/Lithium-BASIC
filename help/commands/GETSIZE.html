GETSIZE name, widthVar, heightVar


Description:

Retrieves the width and height of the window or control specified by name and places the values into the variables specified by widthVar and heightVar.


Usage:

window "win", "GetSize Test", normal, 100, 100, 500, 400
control "btn1", "win", "GetSize", button, 200, 175, 100, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     var w, h as number
     GetSize "win", w, h
     message w & " x " & h, "Current Size"
end sub
