REDIM arrayName(maxIdx1 [,maxIdx2])


Description:

Changes the dimensions of an array created using the ARRAY or DIM commands. All the items in an array will be deleted when it is redimensioned.


Usage:

var num, x as number
var tmpName as string
array nameList(0) as string

input "Enter number of names: ", num
ReDim nameList(num)

for x = 1 to num
     input "Enter name " & x & ": ", tmpName
     nameList(x) = tmpName
next x

print

for x = 1 to num
     print nameList(x)
next x
