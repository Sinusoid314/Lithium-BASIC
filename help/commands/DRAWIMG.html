DRAWIMG winName, imgName, x, y


Description:

Draws an image on the window or control specified by winName. The image specified by imgName must have been loaded first using the LOADIMG command. The x and y parameters give the position at which the image will be drawn.


Usage:

loadimg "pic1", DefPath & "\examples\image1.bmp"

window "win", "Graphics Test", normal, 100, 100, 600, 400
control "draw1", "win", "", drawbox, 100, 50, 300, 300

DrawImg "draw1", "pic1", 50, 50

pause
