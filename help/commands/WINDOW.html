WINDOW name, caption, type, x, y, width, height


Description:

Creates a window labeled name> The window's caption text is specified by caption, its position by x and y, and its size by width and height. The type parameter is an identifier specifying the window's type, and can be any one of the following:

normal - A normal, resizable window with minimize, maximize, and close buttons.
dialog - A non-resizable window with only a close button.
dialog_modal - A dialog window that disables the window below it until it closes.
popup - A non-resizable window with no caption and no border.


Usage:

Window "win", "Window Test", normal, 100, 100, 500, 400

event "win", "close", WinClose

pause

sub WinClose
     var a as string
     question "Are you sure you want to quit?", "Window Test", a
     if a = "yes" then closewindow "win"
end sub
