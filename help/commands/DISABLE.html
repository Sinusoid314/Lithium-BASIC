DISABLE name


Description:

Disables the window or control specified by name. Use the ENABLE command to re enable it.


Usage:

window "win", "Disable Test", normal, 100, 100, 500, 400
control "btn1", "win", "Disable Me", button, 200, 100, 150, 25

event "btn1", "click", DisableClick

pause

sub DisableClick
     Disable "btn1"
end sub
