LINE name, x1, y1, x2, y2


Description:

Draws a line on the window or control specified by name starting at point x1, y1 and ending at point x2, y2. The line is drawn using the window's current forecolor (specified in the FORECOLOR command) and the current line size (specified in the LINESIZE command.)


Usage:

window "win", "Line Test", normal, 100, 100, 500, 400

Line "win", 10, 10, 10, 100
Line "win", 10, 100, 100, 100
Line "win", 100, 100, 100, 10
Line "win", 100, 10, 10, 10

pause
