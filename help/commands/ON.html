ON value GOTO/GOSUB @label1, @label2, ...


Description:

Jumps to a certain branch label based on value. If value 1, for example, then the code will jump to @label1. If it is 2, then the second branch label will be jumped to, and so on. If value is greater than or less than the number of branch labels given, the command will just be skipped. Specify either GOTO or GOSUB depending on whether you want to return from the jump or not.


Usage:

showconsol
var labelNum as number

input "Please choose a branch label: ", labelNum
On labelNum GoSub @one, @two, @three
print "Return"
end

@one
print "Label One"
return

@two
print "Label Two"
return

@three
print "Label Three"
return
