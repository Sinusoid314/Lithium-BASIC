SETFONT winName, fontName, fontSize


Description:

Sets the font of any text drawn on the window or control specified by winName. If fontName does not match any of the installed fonts, the closest font will be chosen. fontSize specifies both the width and height of each character drawn with the font.


Usage:

window "win", "Font Test", normal, 100, 100, 500, 400

SetFont "win", "times_new_roman", 20
drawtext "win", "ALL YOUR BASE ARE BELONG TO US", 10, 10
SetFont "win", "westminster", 50
drawtext "win", "SOMEBODY SET UP US THE BOMB", 10, 100

pause
