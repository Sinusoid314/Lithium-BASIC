UNBINDVAR var1 FROM var2


Description:

Unbinds var1 from var2. For this to happen, var1 must have already been bound to var2 using the BINDVAR command. Once var1 has been unbound, var2's value can be changed without the changes affecting var1.


Usage:

showconsol

var v1, v2 as string

bindvar v1 to v2
v2 = "one"

print v1
print v2
print

UnBindVar v1 From v2
v2 = "two"

print v1
print v2


Produces:
one
one

one
two
