GETDIRS path, listArray(


Description:

Obtains a list of all the folders within the directory specified by path and places the list into a string array specified by listArray(. The array will automatically be resized to a single-dimensioned array long enough to hold the folder list. The 0 index will hold the number of folders in the list, and the other items in the array will contain the actual list. Use the GETFILES comand to obtain a list of files in a certain directory.


Usage:

showconsol

array dirs(0) as string
var n as number
var path as string

input "Enter a directory: ", path

GetDirs path, dirs(

for n = 1 to dirs(0)
     print dirs(n)
next n
