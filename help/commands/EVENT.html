EVENT name, event, subName


Description:

Allows a piece of program code to be executed when an event is generated by the window or control specified by name. The parameter event is a string that specifies the event. Some events include button clicks (specified by a "click" string) and mouse movements over the window or control. The code to execute is specified by subName, which can be the name of any Sub or Function (without any arguments). Refer to the control descriptions in the GUI section for a list of what events each control generates.


Usage:

window "win", "Control Event", normal, 100, 100, 500, 400
control "btn1", "win", "Button 1", button, 200, 150, 100, 25

Event "btn1", "click", ClickSub

pause

sub ClickSub
     message "Button 1 was clicked", "Click Event"
end sub
