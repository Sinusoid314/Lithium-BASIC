ADDITEM name, item [,itemIdx]


Description:

Adds a new item, specified by the string item, to the listbox or combobox control specified by name. If itemIdx is specified, then the new item will be placed at that position in the list, instead of at the end of it.


Usage:

window "win", "Listbox Test", normal, 100, 100, 500, 350
control "list1", "win", "", listbox, 100, 80, 150, 200

AddItem "list1", "One"
AddItem "list1", "Two"
AddItem "list1", "Three"

message GetItem("list1", 2), ""
