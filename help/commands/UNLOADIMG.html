UNLOADIMG name


Description:

Unloads an image that was loaded using the LOADIMG command.


Usage:

loadimg "img1", DefPath & "\examples\image1.bmp"

window "win", "Image Test", normal, 100, 100, 400, 300

event "win", "close", WinClose
drawimg "win", "img1", 100, 100

pause

sub WinClose
     UnloadImg "img1"
end sub
