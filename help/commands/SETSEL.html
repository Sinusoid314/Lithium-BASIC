SETSEL name, startPos, endPos


Description:

Selects a section of text in the textbox or texteditor control specified by name. The values startPos and endPos specify the starting and ending positions of the text to select.


Usage:

window "win", "Select Text", normal, 100, 100, 400, 400
control "text1", "win", "abcdefghi", texteditor, 100, 50, 200, 150
control "btn1", "win", "Set Selection", button, 100, 250, 120, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     SetSel "text1", 3, 5
end sub
