SETSELTEXT name, text


Description:

Replaces the currently selected text with text in the textbox or texteditor contorl specified by name. If there is no text selected, then text will be inserted at the text cursor's position.


Usage:

window "win", "Replace Text", normal, 100, 100, 400, 400
control "text1", "win", "", texteditor, 100, 50, 200, 150
control "btn1", "win", "Replace", button, 100, 250, 100, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     SetSelText "text1", "****"
end sub
