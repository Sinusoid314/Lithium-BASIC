CLEAR name


Description:

Clears the window or control specified by name of all the graphics that have been drawn on it, whether or not they have been stuck to the window using the STICK command.


Usage:

window "win", "Graphics Test", normal, 100, 100, 600, 400
control "draw1", "win", "", drawbox, 100, 50, 300, 300
control "btn2", "win", "Clear", button, 450, 150, 80, 25

event "btn2", "click", ClearWin

drawtext "draw1", "Beam me up", 50, 100
stick "draw1"

pause

sub ClearWin
     Clear "draw1"
end sub
