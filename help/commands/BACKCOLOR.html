BACKCOLOR name, colorVal


Description:

Sets the color to use for filling in shapes drawn on a window or control. name specifies the window or control, and colorVal specifies the number representation of the color. There are system variables available that contain the number values for the basic colors.


Usage:

window "win", "Graphics Test", normal, 100, 100, 600, 400
control "draw1", "win", "", drawbox, 100, 50, 300, 300
Backcolor "draw1", Orange
box "draw1", 10, 10, 100, 100
pause
