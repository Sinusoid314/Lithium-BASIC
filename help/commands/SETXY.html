SETXY name, x, y


Description:

Sets the x and y position of the window or control specified by name.


Usage:

window "win", "SetXY Test", normal, 100, 100, 500, 400
control "btn1", "win", "Set Position", button, 100, 100, 150, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     SetXY "win", 0, 0
end sub
