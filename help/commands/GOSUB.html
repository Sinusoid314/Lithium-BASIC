GOSUB @branchLabel


Description:

Jumps to a section of code defined by @branchLabel, like the GOTO command, except the flow of control can jump back when a RETURN statement is reached.


Usage:

showconsol
var n as number

n = 1
GOSUB @printMulti
n = 2
GOSUB @printMulti
n = 3
GOSUB @printMulti

end

@printMulti
print a*1
print a*2
print a*3
print
return

3
