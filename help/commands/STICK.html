STICK name


Description:

Causes all the graphics that have been drawn to the window or control specified by name to stick. This keeps the graphics from being erased when another window overlaps them or the window is refreshed using the REFRESH command.


Usage:

loadimg "img1", DefPath & "\examples\image1.bmp"

window "win", "Refresh Test", normal, 100, 100, 500, 400
control "draw1", "win", "", drawbox, 10, 10, 100, 100
control "btn1", "win", "Refresh", button, 100, 300, 100, 25

drawimg "draw1", "img1", 10, 10
Stick "draw1"
drawimg "draw1", "img1", 40, 40

event "btn1", "click", RefreshWin

pause

sub RefreshWin
     refresh "draw1"
end sub
