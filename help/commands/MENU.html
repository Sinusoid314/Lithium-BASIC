MENU winName, menuText, itemText1, subName1, |, itemText2, subName2, ...


Description:

Creates a menu on the window specified by winName. The text that is displayed on the menu bar is specified by menuText. Following this are pairs of parameters that specify the text and event sub or function of each item in the menu. The parameters itemText1 and subName1 create the first menu item, itemText2 and subName2 create the second item, and so on. To add a menu item seperator, just place a | character as a parameter in between these pairs. Adding a & before a character in an item's text will underline that character.


Usage:

window "win", "Menu Test", normal, 100, 100, 500, 400
Menu "win", "Menu 1", "Item 1-1", Item11Click, "Item 1-2", _
         Item12Click, |, "Item 1-3", Item13Click
Menu "win", "Menu 2", "Item 2-1", Item21Click

pause

sub Item11Click
     message "Item 1 of Menu 1 was clicked", ""
end sub

sub Item12Click
     message "Item 2 of Menu 1 was clicked", ""
end sub

sub Item13Click
     message "Item 3 of Menu 1 was clicked", ""
end sub

sub Item21Click
     message "Item 1 of Menu 2 was clicked", ""
end sub
