MID stringVar, start [,length], replaceStr


Description:

Replaces a portion of the value of string variable stringVar with the string replaceStr, starting at start. If length is specified, then only that amount of characters will be replaced in stringVar.


Usage:

var ss as string
var n as number

showconsol

input "Enter a string: ", ss

for n = 1 to len(ss) step 2
     Mid ss, n, "*"
next n
