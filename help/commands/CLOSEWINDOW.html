CLOSEWINDOW name


Description:

Closes the window specified by name. The window must have been created first using then WINDOW command. The program will end if the last window is closed and the consol window is no longer visible.


Usage:

window "win", "Window Test", normal, 100, 100, 500, 400
control "closeBtn", "win", "Close this window", button, 200, 100, 100, 25

event "closeBtn", "click", BtnClick

pause

sub BtnClick
     CloseWindow "win"
end sub
