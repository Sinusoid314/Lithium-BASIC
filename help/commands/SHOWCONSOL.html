SHOWCONSOL


Description:

Displays the consol window. The consol window is initially hidden when a program starts. Use the HIDECONSOL command to hide it.


Usage:

window "win", "Using the Consol", normal, 100, 100, 400, 300
control "btn1", "win", "Consol", button, 100, 100, 100, 25

event "btn1", "click", UseConsol

pause

sub UseConsol
     var cap as string
     ShowConsol
     input "Enter new caption: ", cap
     settext "btn1", cap
     hideconsol
end sub
