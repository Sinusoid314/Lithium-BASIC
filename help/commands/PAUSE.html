PAUSE


Description:

Pauses program execution and waits for window events. This command is necessary for programs using GUI windows, otherwise the program will not wait for events and would end prematurely.


Usage:

window "win", "GetXY Test", normal, 100, 100, 500, 400
control "btn1", "win", "GetXY", button, 200, 175, 100, 25

event "btn1", "click", Btn1Click

Pause

sub Btn1Click
     var left, top as number
     getxy "win", left, top
     message "(" & left & ", " & top & ")", "Current Position"
end sub
