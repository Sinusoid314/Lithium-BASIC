BOX name, x, y, width, height


Description:

Draws a box onto a window or control specified by name, using its current background and foreground colors. The x and y specify the position the box should be drawn at, and width and height specify its size.


Usage:

window "win", "Graphics Test", normal, 100, 100, 600, 400
control "draw1", "win", "", drawbox, 100, 50, 300, 300

box "draw1", 10, 10, 100, 100

pause
