BUTTONIMG btnName, picName


Description:

Changes the image displayed on the picbutton control specified by btnName. The image specified by picName must have been loaded first by using the LOADIMG command.


Usage:

loadimg "get", DefPath & "\examples\get.bmp"

window "win", "Control Test", normal, 100, 100, 600, 400
control "pbtn1", "win", "", picbutton, 100, 100, 72, 25

ButtonImg "pbtn1", "get"

pause
