QUESTION message, title, answerVar


Description:

Displays a message box, with the given message and title, that gives the user a "Yes" or "No" choice. The choice is placed in answerVal, which can either be "yes" or "no".


Usage:

window "win", "Question Test", normal, 100, 100, 500, 400

event "win", "close", WinClose

pause

sub WinClose
     var aVar as string
     Question "Are you sure you want to quit?", "Closing...", aVar
     if aVar = "yes" then closewindow "win"
end sub
