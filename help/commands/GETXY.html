GETXY name, xVar, yVar


Description:

Retrieves the x and y position of the window or control specified by name and places the values into the variables specified by xVar and yVar.


Usage:

window "win", "GetXY Test", normal, 100, 100, 500, 400
control "btn1", "win", "GetXY", button, 200, 175, 100, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     var left, top as number
     GetXY "win", left, top
     message "(" & left & ", " & top & ")", "Current Position"
end sub
