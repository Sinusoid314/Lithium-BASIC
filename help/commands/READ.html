READ var1, var2, ...


Description:

Reads in a number of data items that were defined by the DATA command. The amount of data items read is based on the number of variables specified. If a data item's type does not match the type of the corrisponding variable, it will be converted to fit. To start reading data items from the beginning, use the RESTORE command.


Usage:

data 4
data "Andrew", "Dan", "Vince", "Trish"

showconsol

var nameNum, n as number
var nameStr as string

Read nameNum

for n = 1 to nameNum
     Read nameStr
     print nameStr
next n
