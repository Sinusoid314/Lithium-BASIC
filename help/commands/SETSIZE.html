SETSIZE name, width, height


Description:

Sets the width and height of the window or control specified by name.


Usage:

window "win", "SetSize Test", normal, 100, 100, 500, 400
control "btn1", "win", "Resize", button, 50, 50, 100, 25

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     SetSize "win", 200, 200
end sub
