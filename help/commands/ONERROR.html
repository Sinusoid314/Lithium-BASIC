ONERROR command


Description:

Executes a Lithium BASIC command, specified by command, whenever a runtime error occurs. This allows the program to continue executing without an error message being displayed.


Usage:

showconsol
array arrVals(3) as string
var inVal as string
var n as number

OnError GOTO @err

@enterVal
input "Enter an item index: ", n
input "Enter the item's value: ", inVal
arrVals(n) = inVal
goto @enterVal

@err
print "Index out of bounds"
end
