SETSOUNDPOS soundName, position


Description:

Sets the position to start play from in the sound specified by soundName. The position can be any number between 0 and the length of the sound. To get the sound's length, use the GETSOUNDLEN() function.


Usage:

var v as string
var musicLen as number

showconsol

opensound "mid", DefPath & "\examples\the guess who - no time.mid"
playsound "mid"

musicLen = getsoundlen("mid")

@skipPrompt
input "Press enter ('q' to quit): ", v
if v = "q" then goto @done
call ResetSound
goto @skipPrompt

@done
closesound "mid"

sub ResetSound
     var musicPos as number
     musicPos = int(rnd()*musicLen)+1
     SetSoundPos "mid", musicPos
end sub
