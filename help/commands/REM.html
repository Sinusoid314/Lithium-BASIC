REM remark


Description:

Adds a remark to a rpogram's code. Any REM lines are cut out by the compiler, and do not effect the program's function in any way. Remarks are placed in code to explain parts of the code in plain English. An easier way of placing a remark in your code is to use a ' instead of REM.


Usage:

REM this is a remark
' this is also a remark
