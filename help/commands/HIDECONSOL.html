HIDECONSOL


Description:

Hides the consol window. The consol window is initially hidden when a program starts, and must be shown first using the SHOWCONSOL command.


Usage:

window "win", "Using the Consol", normal, 100, 100, 400, 300
control "btn1", "win", "Consol", button, 100, 100, 100, 25

event "btn1", "click", UseConsol

pause

sub UseConsol
     var cap as string
     showconsol
     input "Enter new caption: ", cap
     settext "btn1", cap
     HideConsol
end sub
