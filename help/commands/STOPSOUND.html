STOPSOUND name


Description:

Stops play of the sound specified by name. The sound must have already been loaded using the OPENSOUND command.


Usage:

showconsol
var n as number

opensound "song", DefPath & "\examples\the guess who - no time.mid"
print "(3)Play (2)Pause (1)Stop (0)Quit"

@choose
input ">> ", n
if n = 3 then playsound "song"
if n = 2 then pausesound "song"
if n = 1 then StopSound "song"
if n = 0 then goto @done
goto @choose

@done
closesound "song"
