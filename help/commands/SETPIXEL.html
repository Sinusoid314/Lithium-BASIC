SETPIXEL name, x, y, color


Description:

Changes the color of the pixel at position x, y on the window or to control name to the color specified by color. There are system variables available that contain the number values for the basic colors.


Usage:

var a, b as number

window "win", "SetPixel Test", normal, 100, 100, 400, 400

for a = 100 to 150
     for b = 100 to 150
          SetPixel "win", a, b, Blue
     next b
next a

pause
