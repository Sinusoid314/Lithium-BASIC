SETSTATE name, state


Description:

Sets the state of the radiobutton or checkbox specified by name. The state is a number that specifies whether the control is checked (True or 1) or unchecked (False or 0).


Usage:

window "win", "SetState Test", normal, 100, 100, 500, 400
control "rbtn", "win", "Radio Button", radiobutton, 150, 50, 110, 20
control "cbox", "win", "Check Box", checkbox, 150, 90, 100, 20
control "btn1", "win", "Check", button, 150, 200, 150, 25
control "btn2", "win", "Uncheck", button, 150, 230, 150, 25

event "btn1", "click", Btn1Click
event "btn2", "click", Btn2Click

pause

sub Btn1Click
     SetState "rbtn", True
     SetState "cbox", True
end sub

sub Btn2Click
     SetState "rbtn", False
     SetState "cbox", False
end sub
