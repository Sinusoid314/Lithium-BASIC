DELITEM name, index


Description:

Removes an item from a listbox or combobox control specified by name. The index parameter specifies the index of the item to be removed.


Usage:

window "win", "DelItem Test", normal, 100, 100, 500, 400
control "btn1", "win", "Delete", button, 200, 250, 100, 25
control "list1", "win", "", listbox, 200, 100, 100, 100

additem "list1", "One"
additem "list1", "Two"
additem "list1", "Three"

event "btn1", "click", Btn1Click

pause

sub Btn1Click
     DelItem "list1", 1
end sub
