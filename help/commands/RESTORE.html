RESTORE [position]


Description:

Restores the READing of data items to the item specified. If no item number is specified, the reading is restored to the first data item.


Usage:

data 4
data "Andrew", "Dan", "Vince", "Trish"

showconsol

var nameNum, n as number
var nameStr as string

read nameNum

for n = 1 to nameNum
     read nameStr
     print nameStr
next n

input "Choose a name to print: ", n
Restore n+1
read nameStr
print nameStr
