Combo Box

A combo box control is made up of both a text box section and a listbox section. The listbox section only gets displayed when the user presses the arrow button next to the text box.

Events:

Select - When the user clicks once on a list item.
DoubleSelect - When the user double clicks on a list item.
Change - When the text inside the text box section gets changed.

Commands:

AddItem
DelItem
SetItem
SetSelIdx


Functions:

GetSelIdx()
GetItem()
ItemCount
