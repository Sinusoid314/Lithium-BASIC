Windows

Windows are frames that allow the user to interact with a Windows program through controls that are placed on them. Note: All the events, commands, and functions listed below can also be used with just about every type of control, unless otherwise noted.

Events:

Close (Window-specific) - When the user tries to close a window.
LeftButtonUp - When the left mouse button is released after being pressed.
LeftButtonDown - When the left mouse button is pressed.
LeftButtonDouble - When the left mouse button is double-clicked.
RightButtonUp - When the right mouse button is released after being pressed.
RightButtonDown - When the right mouse button is pressed.
RightButtonDouble - When the right mouse button is double-clicked.
MouseMove - When the mouse cursor moves over the window or control.
KeyDown - When a key is pressed while the window or control has focus.
KeyUp - When a key is released while the window or control has focus.
Paint - When the window or control needs to be redrawn.
Resize - When the window or control is resized.
