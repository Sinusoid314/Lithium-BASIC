ITEMCOUNT(name)


Description:

Returns the number of items in the listbox or combobox specified by name.


Usage:

window "win", "Listbox Test", normal, 100, 100, 500, 350
control "list1", "win", "", listbox, 100, 80, 150, 200

AddItem "list1", "One"
AddItem "list1", "Two"
AddItem "list1", "Three"

message ItemCount("list1"), ""
