GETITEM(name, index)


Description:

Returns the text of a listbox or combobox control specified by name at the item number index.


Usage:

window "win", "Listbox Test", normal, 100, 100, 500, 350
control "list1", "win", "", listbox, 100, 80, 150, 200

AddItem "list1", "One"
AddItem "list1", "Two"
AddItem "list1", "Three"

message GetItem("list1", 2), ""
