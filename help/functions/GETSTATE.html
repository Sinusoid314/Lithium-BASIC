GETSTATE(name)


Description:

Gets the current state of a radiobutton or checkbox control specified by name. A 1 is returned if the control is checked; otherwise, 0 is returned.


Usage:

window "win", "Control Test", normal, 100, 100, 600, 400
control "btn1", "win", "Get State", button, 100, 100, 100, 25
control "cbox1", "win", "Check Box", checkbox, 100, 200, 100, 15

event "btn1", "click", IsChecked

pause

sub IsChecked
message GetState("cbox1"), ""
end sub
