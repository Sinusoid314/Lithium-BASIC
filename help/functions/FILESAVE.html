FILESAVE(title, filter)


Description:

Opens a file selection dialog box with a "Save" button. The title parameter is a string that holds the caption for the window, and filter tells the dialog box what types of files to show. The filter string must contain both a description and filter pattern, seperated by a |. For example, when opening a text file with FILESAVE, the filter string "Text files | *.txt" may be used, where "Text files" is the description and "*.txt" is the filter pattern. Multiple file types may be specified by seperating each type with a ; in the pattern string, as in "Code files | *.bas;*.bak;*.txt". The return value is a string with the filename of the selected file.


Usage:

showconsol
print "FileSave - " & FileSave("Save...","Code Files (*.txt,*.bas,*.bak) | *.txt;*.bas;*.bak")
