RGB(red, green, blue)


Description:

Returns the color value that is made up from the red, green, and blue values given.

function RandColor() as number
     var r, g, b as number
     r = int(rnd()*255)+1
     g = int(rnd()*255)+1
     b = int(rnd()*255)+1
     RandColor = RGB(r, g, b)
end function
