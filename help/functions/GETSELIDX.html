GETSELIDX(name)


Description:

Returns the index number of the selected item in a listbox or combobox specified by name.


Usage:

window "win", "Listbox Test", normal, 100, 100, 500, 350
control "list1", "win", "", listbox, 100, 80, 150, 200

event "list1", "leftbuttonup", getSel
AddItem "list1", "One"
AddItem "list1", "Two"

pause

sub getSel
message GetSelIdx("list1"), ""
end sub
